## 🚀 Sobre o Projeto

O **Fokus** é um cronômetro baseado na técnica Pomodoro, criado para ajudar na organização do tempo e foco. O grande diferencial aqui foi o uso de **JavaScript puro** para manipular o comportamento da página sem a necessidade de frameworks, focando em performance e lógica sólida.

Este projeto demonstra meu conhecimento em **Engenharia de Software** e minha transição para o desenvolvimento Full-Stack.

## 🎨 Diferenciais do Desenvolvimento

* **Manipulação de Áudio**: Implementação de sons de alerta e música de fundo utilizando a API de áudio nativa do navegador.
* **Contextos Dinâmicos**: O fundo da tela e os textos mudam via JavaScript ao alternar entre Foco, Descanso Curto e Longo.
* **Layout Impecável**: Aplicação das mesmas correções de largura que usamos no Jornada Viagens, garantindo que o `bottom-hero` e o `footer` ocupem toda a tela.

## 📦 Como rodar o projeto

1. Baixe os arquivos do repositório.
2. Abra o arquivo `index.html` diretamente no seu navegador ou utilize a extensão **Live Server** no VS Code.
